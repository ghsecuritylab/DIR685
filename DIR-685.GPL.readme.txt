
o This file contains steps for untar toolchain and building the image.

1.untar the source code
	tar zxvf dir685.gpl.source.tgz
	
2. steup the toolchain

	~$ su
	~$ Password:
	~$ cd dir685
	~$ cp -rf storlink_arm_920t_le_uclibc.20080713 /opt/storlink_arm_920t_le_uclibc.20080713
	~$ cd /opt && ln -s storlink_arm_920t_le_uclibc.20080713 storlink_arm_920t_le_uclibc

3. You can use the toolchain to build image now.

	Building the image
	~$ change directory to dir685 source 
	~$ source boards/wrgns01/setup_env 
	~$ make
	~$ make

  After make successfully, you will find the image file in ./images/.

  Upgrade the new firmware by web interface provide by device, 
  then you will get your specific image.


NOTE: If you get syntax error message when you building source please downgrade your make version to 3.80.
